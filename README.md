# SOC Incident Analysis Portfolio

## Overview

This repository contains documented incident investigations performed in a simulated Security Operations Center (SOC) environment.

The purpose of this portfolio is to demonstrate:

- Security event investigation methodology
- SIEM log analysis (Splunk & Elastic)
- Windows Event Log analysis
- RDP and VPN activity investigation
- MITRE ATT&CK mapping
- Incident reporting and documentation standards
- Analytical reasoning and false positive validation

---

## Tools & Technologies

- Splunk (SPL)
- Elastic Stack (KQL)
- Windows Security Event Logs
- VPN Log Analysis
- MITRE ATT&CK Framework
- Threat Intelligence Validation

---

## Investigation Methodology

Each case follows a structured workflow:

1. Alert identification
2. Log correlation
3. Event validation
4. Behavioral analysis
5. Threat classification
6. MITRE mapping
7. Risk assessment
8. Final determination

---

## Case Studies

### VPN Brute Force Analysis
Investigation of abnormal VPN connection spikes and identification of source IP responsible for authentication attempts.

### RDP Logon Investigation
Analysis of Windows Logon Types and detection of potential lateral movement activity.

### Phishing Case Study
Email header analysis, authentication verification (SPF/DKIM/DMARC), and malicious intent assessment.
